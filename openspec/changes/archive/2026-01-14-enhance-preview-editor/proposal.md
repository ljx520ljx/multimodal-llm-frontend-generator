# Proposal: Phase 4 - 交互预览 & 编辑器增强

## Why

Phase 3 已完成前端基础框架（双模式布局、Sandpack 预览、Monaco 编辑器），但存在以下待完善功能：

1. **编辑器-预览未联动**：手动编辑代码后，预览不会自动更新
2. **上传未对接后端**：图片上传仅本地预览，未调用后端 API
3. **预览配置不完善**：Sandpack 配置需优化以支持更好的交互体验
4. **生成状态未完善**：生成中的 loading/进度展示需增强
5. **环境变量未配置**：需要 `.env.local` 连接后端

## What Changes

### 1. 编辑器-预览实时联动
- 编辑代码时预览自动更新（防抖处理）
- 预览错误时提供友好提示和恢复选项

### 2. 图片上传对接后端
- 调用 `POST /api/upload` 上传图片
- 处理上传状态（进度、错误）
- 存储返回的 imageIds 用于后续生成

### 3. 代码生成完整流程
- 调用 `POST /api/generate` 获取 SSE 流
- 展示思考过程（thinking）
- 流式更新代码和预览

### 4. 预览体验优化
- 添加预览加载状态
- 添加刷新按钮
- 添加全屏预览模式
- 优化错误边界处理

### 5. 开发者模式增强
- 编辑器语言切换（TSX/CSS）
- 代码格式化按钮
- 复制代码功能
