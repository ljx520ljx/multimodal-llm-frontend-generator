## 验收标准

### 编辑器-预览联动
- [ ] 在开发模式下编辑代码，预览在 300ms 后自动更新
- [ ] 代码有语法错误时，预览显示友好错误提示
- [ ] 错误修复后预览自动恢复

### 预览体验
- [ ] 可点击刷新按钮手动刷新预览
- [ ] 可点击全屏按钮进入全屏预览
- [ ] 全屏模式下可正常交互，ESC 退出
- [ ] 预览加载时显示骨架屏

### 编辑器功能
- [ ] 可切换编辑 App.tsx 或 styles.css
- [ ] 可复制当前代码到剪贴板
- [ ] JSX 语法高亮正确

### 完整流程
- [ ] 上传图片 → 显示上传进度 → 调用后端
- [ ] 点击生成 → 显示思考过程 → 流式更新代码
- [ ] 代码生成完成 → 预览自动渲染 → 可交互
