# Proposal: add-chat-style-input

## Summary

重构左侧面板为统一的聊天式交互界面，整合图片上传和对话修改功能，提供类似 ChatGPT/Claude 的交互体验。

## Motivation

当前界面存在以下问题：
1. **交互分散**：图片上传在左侧，对话修改在右侧，用户需要切换关注区域
2. **操作不统一**：拖拽上传和对话修改是两个独立流程
3. **不符合现代 AI 产品交互习惯**：ChatGPT/Claude 等产品都采用统一的聊天式输入

## Goals

1. **整合对话修改到左侧面板** - 移除右侧独立的对话修改区
2. **统一输入方式** - 拖拽图片、粘贴图片、文字输入都在同一个输入区域
3. **ChatGPT 风格的输入框** - 输入框 + 右侧发送按钮
4. **对话历史展示** - 显示用户输入和 AI 响应的历史记录

## Non-Goals

- 不改变代码编辑器和预览区域的布局
- 不修改后端 API 结构

## Scope

### Affected Capabilities
- `image-upload`: 新增剪切板粘贴功能
- `chat-interaction`: 移动到左侧面板
- `ui-layout`: 移除右侧对话面板

### New Components
- `UnifiedInput`: 统一的聊天式输入组件（文字 + 图片 + 发送按钮）
- `ChatHistory`: 对话历史展示组件

### Removed Components
- 右侧独立的 `ChatPanel` 将被整合到左侧

## New Layout

```
┌─────────────────────────────────────────────────────────────┐
│                    左侧面板 (重构)                            │
├─────────────────────────────────────────────────────────────┤
│  拖拽上传区域 (点击或拖拽添加图片)                            │
├─────────────────────────────────────────────────────────────┤
│  已上传图片列表 (可拖拽排序、删除)                            │
├─────────────────────────────────────────────────────────────┤
│  对话历史 (生成后显示)                                       │
│  ├─ 用户: [3张图片] 生成一个登录页面                         │
│  ├─ AI: 已生成原型...                                       │
│  ├─ 用户: 把按钮改成蓝色                                    │
│  └─ AI: 已更新...                                          │
├─────────────────────────────────────────────────────────────┤
│  ┌────────────────────────────────┐ ┌──────────────────┐   │
│  │ 输入需求，支持粘贴图片...       │ │ 生成/发送 按钮   │   │
│  └────────────────────────────────┘ └──────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## Risks

- **布局变化较大**：用户需要适应新界面
  - 缓解：保持核心功能不变，只是布局整合

## Success Criteria

1. 用户可在输入框粘贴图片，与拖拽上传效果相同
2. 生成和对话修改都通过同一个输入框完成
3. 对话历史在左侧面板显示
4. 右侧对话面板被移除
